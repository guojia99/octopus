# octopus
> - 这是一个拥有各种各样基于基础库开发的一个工具库
> - 本仓库基于go1.18开发
> - 本仓库不依赖其他第三方库

# 项目列表

### 1、OMap

- [`omap`](./omap): 基于`list.List` 封装的一个map, 支持按key排序，支持拓展的map相关函数，支持锁,

- `Map` 为any原型
- `OMap`为泛型
- 使用方法参见 [`map example`](./_example/map) 



### 2、OArray 

- [`oarray`](./oarray):  带锁的数组列表，包含一些实用方法
- `Array` 为any 原型
- `OArray` 为泛型
